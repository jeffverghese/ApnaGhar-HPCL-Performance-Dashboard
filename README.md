# Apna Ghar HPCL Performance Dashboard

## Overview
Developed an interactive Power BI dashboard to analyze operational performance of Apna Ghar HPCL outlets across Haryana.

## Key Metrics
- Total Apna Ghar Outlets
- Total Bookings
- Revenue Generated
- Average Revenue per Booking

## Dashboard Features
- Top Performing Apna Ghar Analysis
- City-wise Contribution Analysis
- Revenue by Outlet
- Monthly Booking Trends
- Interactive Filtering

## Tools Used
- Power BI
- Excel
- DAX
- Data Visualization
